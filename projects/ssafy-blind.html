SSAFY BLIND

SSAFY 교육생만 참여할 수 있는 익명 커뮤니티 플랫폼입니다. 인증 흐름, 예외 처리, 보안 설정, 프론트엔드 연동까지 개인 프로젝트로 구현했습니다.

핵심 구현

  • Mattermost API를 프록시해 SSAFY 교육생 인증 흐름을 구성했습니다.
  • JWT를 HttpOnly Cookie로 관리해 클라이언트 노출 위험을 줄이는 인증 구조를 적용했습니다.
  • @RestControllerAdvice 기반 글로벌 예외 처리로 API 응답 형식을 일관되게 정리했습니다.
  • Spring Security 인증 필터와 AOP 기반 트랜잭션 흐름을 디버깅하며 동작 원리를 검증했습니다.

배운 점

단순 CRUD보다 인증과 보안 흐름이 서비스 신뢰도에 더 큰 영향을 준다는 점을 체감했습니다. 특히 쿠키 기반 JWT와 Spring Security 필터 체인을 연결하면서 요청이 컨트롤러에 도달하기 전 어떤 순서로 검증되는지 명확히 이해하게 되었습니다.
